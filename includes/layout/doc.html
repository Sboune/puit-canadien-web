Documentation

L'application web

Le but de cette application web est l'analyse de données géothermique. A travers une interface claire et simple d'utilisation, vous pourez analyser un ensemble de donnée prealablement acquise, grâce à differents outils( graphique, tableau de statistique )

Les vues 3D

Se deplacer

Pour se deplacer dans la vue 3D, vous devez utiliser la souris. Le clic gauche permet de faire tourner l'ensemble de la vue 3D autour du points centrale. Le clic droit quant a lui permet de deplacer la vue lateralement.

Zoomer

Pour zommer dans la vue 3D, vous devez utiliser la molette. Pour zoomer en avant il faut faire tourner la roulette vers le haut ou appuier sur le bouton "Zoom +". Pour dezoomer il faut faire tourner la roulette vers le bas ou appuier sur le bouton "Zoom -". Pour reinitialiser le zoom cliquez sur bouton "reset".

Selection

Les vues 3D represent les dispositifs reels. Grace a ces vue 3D vous pouvez selectioner les capteurs dont vous voulez anyliser les données. Pour selectioner un capteur, il suffit de cliquer dessus. Pour selectioner plusieurs capteurs, cliquez sur l'ensemble des capteurs que vous voulez selectioner. Les capteurs selectionné s'ajout automatiquement au graphique et au tableau de statistique. Pour deselectioner un capteur il suffit de recliquer dessus.
